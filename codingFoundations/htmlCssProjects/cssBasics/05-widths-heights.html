Width and Height Notes

The width and height properties are used to et the width and height of an element.

The default setting for width and height is auto, which means the browser decides the width and height.

Other settings for width and height are:

  • Specified length values such as px and em
  • Percentage of the containing element

This element has a width of 50% and a height of 200px

You can also manipulate width and height of images.
